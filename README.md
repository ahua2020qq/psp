
  # 程序员工具搜索平台

  This is a code bundle for 程序员工具搜索平台. The original project is available at https://www.figma.com/design/DmlXIvxKnQ0aOPwbhGgDBG/%E7%A8%8B%E5%BA%8F%E5%91%98%E5%B7%A5%E5%85%B7%E6%90%9C%E7%B4%A2%E5%B9%B3%E5%8F%B0.

重要信息：
本项目需要接入 Supabase 做后台服务，实现用户登录鉴权功能，仅 admin 管理员账号可登录后台；
管理员登录后，新增「后台管理入口」，实现：工具站主题配置（明暗模式切换 + 主题色自定义）+ 工具内容批量新增 / 编辑 / 删除 / 分类管理；
普通游客无登录入口，正常访问工具站所有功能，原有页面布局、交互逻辑、暗黑模式完全保留不变。

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
