---
name: analyze-perf
description: 分析项目性能并提供优化建议
---

请执行以下性能分析：

1. **构建分析**
   - 运行 npm run build
   - 分析打包体积
   - 检查是否有大的依赖包

2. **代码分割检查**
   - 检查路由级别的代码分割
   - 检查组件级别的懒加载
   - 报告未使用的代码

3. **缓存效率**
   - 检查 KV Cache 配置
   - 分析缓存命中率（如果有数据）
   - 建议可缓存的端点

4. **API 性能**
   - 检查 Functions 响应时间
   - 分析是否存在 N+1 查询问题
   - 建议优化方向

5. **生成报告**
   - 总结发现的问题
   - 按优先级列出优化建议
   - 提供具体的代码改进建议
